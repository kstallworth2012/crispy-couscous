# Work Management Application - Spring Boot

This is a data driven work and project manager app.
This will allow a user to keep track of projects and files and
goes beyond simple spread sheet tracking.

## Description

## Spring Boot JAVA WORKMANAGER
* There are several key entities that help manage and track the project.
* 1. Projects Entity: This is the overall project object.
* 2. Activities Entity: Activities are specific tasks or project milestones that are part of the project entity.
* 3. Resources Entity: Resources are people and assets that are assigned to activities. A resource can be anything. such as a human resource or some type of equipment used to complete the activity.
* 4. Teams Entity: Groups of resource entities collaborate on particular activities.
* 5. Assignments Entity: this is the resource that is specifically assigned to each activity entity.
* 6. Schedules Entity: This entity serves as both the baseline and actual timeline for activity entities.

## In addition to the CREATE, READ, UPDATE, and DELETE actions for each entity. There will be other actions that users can take.
* 1. LIST ALL Activities and their status.
* 2. Get all ACTIVE PROJECTS AND THEIR MANAGERS.
* 3. VIEW ALL resource allocation for a project
* 4. Track all delayed activities.
